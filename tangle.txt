安裝@iota_sdk 
	安裝 Rust（因為 @iota/client 有原生 binding）
	curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
	source $HOME/.cargo/env
	rustc --version
	cargo --version
	安裝node
	sudo npm install -g typescript
	curl -fsSL https://deb.nodesource.com/setup_22.x | sudo -E bash -
	sudo apt-get install -y nodejs
	node -v
	npm -v

	npm init -y
	# 安裝 Node 類型定義
	npm install --save-dev @types/node

	# 確認 TypeScript 版本 >= 5.0
	npm install --save-dev typescript@latest
	npm install @iota/sdk

生成自己的 Seed & 地址:
	head /dev/urandom | tr -dc 'A-Z9' | head -c81   (產生seed)

	docker run --rm hornet:dev tool seed-to-address \
  --seed "MYSEED9EXAMPLE9FOR9TEST9ONLY9DONT9USE9IN9MAIN9NET9ABCDEFGHIJKLMN9" \
  --index 0   (產生地址)

查交易block內容:
